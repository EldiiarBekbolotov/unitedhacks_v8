# United Hacks V8

Landing site for **United Hacks V8**, Hack United's global virtual hackathon.

## Quick start

```bash
npm install
npm run dev
```

Open [http://localhost:5173](http://localhost:5173).

## Build

```bash
npm run build
npm run preview
```

Static output lands in `dist/` (ready for Render static hosting or any CDN).
